也许很多Linux命令较为抽象不易记忆，但是好记性不如烂笔头，记下来加深记忆，随时补充。

Linux常用命令

pwd

全称：Print Working Directory
查看当前光标所在目录(路径 path)

结果为绝对路径

ls

显示的文件夹和文件的名称

可以看到权限 用户用户组 时间 大小等额外信息

可以显示隐藏文件夹和文件(隐藏文件夹和文件一般为.开头)

仅仅查看文件大小，文件夹不行

按时间升序显示
-r 反转排序顺序
-t 按时间排序(默认降序)

mkdir

创建文件夹

并联创建文件夹

并联创建文件夹，此处-p为递归创建目录

cd

全称：Change Directory
切换目录(路径 path)

切换目标路径

切换到家目录

切换到上一级目录

回退到上一次命令的文件夹(常用)

路径

绝对路径：/根目录开头，路径是齐全的

相对路径：不是以/开头，路径是短的

help

在使用命令的时候有些不熟练，可以通过 –help 去查看如何使用该命令

Usage: mkdir [OPTION]… DIRECTORY…
其中[OPTION] 代表可选项
… 代表可以有多个参数
DIRECTORY 类似单独的大写字母代表必填项

man

相比于help，是更为详细的命令帮助
