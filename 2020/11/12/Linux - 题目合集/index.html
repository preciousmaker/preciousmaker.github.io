日常练习题目

1. 查看当前目录

2. 切换到上一层目录

3. 切换到上一次目录

4. 高危命令有哪些

此命令会覆盖1.log

此命令会强制删除/下的所有文件

5. 文件夹相关的命令有哪些

6. 常用的-R参数的命令

此处的-R意为：处理指定目录以及其子目录下的所有文件

7. vi命令，简述编辑保存流程的快捷键

vi -> INSERT -> ESC -> :wq

8. 绝对路径和相对路径的区别

绝对路径：/根目录开头，路径是齐全的
相对路径：不是以/开头，路径是短的

在编写shell的时候，尽量使用绝对路径

9. 隐藏文件或文件夹如何查看，有何特点

隐藏文件和文件夹都是以.开头

10. 如何查找服务器上，名称含有xx的文件

11. 级联创建文件夹参数是什么

12. 移动和复制的区别，以及是否可以改名

移动最终结果是一份，复制最终结果为两份，均可以改名

13. rwx分别代表数字多少

421

14. rwxr–r-x 数字是多少

745

15. rwxr–r-x 代表什么

文件的所有者具有读写执权限
文件所属组的用户只有读权限
其他组的用户具有读执权限

16. 查看命令帮助，要看什么

–help
要看命令后跟的参数

17. 发现一个用户登录不上或者无法执行命令，是什么文件的问题

/etc/passwd

18. 切换用户带 - ，表示什么

表示切换到该用户并执行该用户的环境变量

19. 临时获取root权限，如何配置？执行要加什么命令？

在/etc/sudoers给用户加上ALL，执行命令前加上sudo

20. vi命令的三种模式

命令、编辑、尾行

21. tail -f 和 tail -F 的区别

-F = -f + retry

22. 管道符是什么命令，过滤是什么命令

| grep
其中grep是正则表达式的意思

23. 别名的命令是什么

alias

24. 个人环境变量使用哪个文件？

.bashrc

25. tab键，按一次和两次，分别是什么含义

一次是补全，两次是显示所有匹配的文件

26. history -c的含义

清空历史命令记录(不留尾巴用的)

27. chmod -R 777 / 这个命令是否允许在生产上执行，为什么

不允许，执行了就等于开放所有文件的所有权限给所有人了

28. 除了ftp sftp上传下载工具，还有什么命令可以实现

rz 上传
sz 下载

29. 使用vi去粘贴内容到文件中，必须在什么模式下，第一行才不会丢失数据？

编辑模式

30. vi清空文件内容，最关键的两个命令是什么

在命令行模式中，gg + dG
gg 跳转到第一行的第一个字母
dG 删除当前行及以下所有行
