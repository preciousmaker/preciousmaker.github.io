#1.下载所需要的安装包
链接：https://pan.baidu.com/s/1gWIBx8A3VQFrjOOwRTBVVg
提取码：wsh1
复制这段内容后打开百度网盘手机App，操作更方便哦

#2.创建hadoop用户
useradd hadoop
su - hadoop

将第一步中的所有安装包放在/home/hadoop/sourcecode

#3.创建一些必要的空的文件夹
mkdir tmp sourcecode software shell log lib data app

#4.安装jdk
略

#5.安装hadoop
cd sourcecode
tar -xzvf hadoop-2.6.0-cdh5.16.2.tar.gz -C ../app/

#6.建立软连接
ln -s hadoop-2.6.0-cdh5.16.2 hadoop

#7.修改为显示指定
cd app/hadoop/etc/hadoop
vi hadoop-env.sh
export JAVA_HOME=/usr/java/jdk1.8.0_181
属于hadoop的一个小bug

#8.修改hosts名
使用root用户
先查看自己的ip
ifconfig
一般是192开头的，例如自己的是192.168.1.100
vi /etc/hosts
在最后一行增加
192.168.1.100 yourname
这里的yourname为自己的名字

#9.配置ssh yourname无密码认证
使用hadoop用户
rm -rf .ssh
ssh-keygen
cd .ssh
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 0600 ~/.ssh/authorized_keys

ssh yourname date
第一次输入会让你再次确定，输入yes，建立关系
正常会返回一个日期

ssh yourname date
再次数据做double check

#10.修改配置文件

#11.启动hadoop
sbin/start-dfs.sh

#12.格式化hdfs
到/home/hadoop/app/hadoop
bin/hdfs namenode -format
便于上传文件到hdfs

#13.hdfs创建文件夹
bin/hdfs dfs -mkdir /user
bin/hdfs dfs -mkdir /user/hadoop

bin/hdfs dfs -put etc/hadoop input
这一步是将etc/hadoop下的文件放到hdfs的/user/hadoop/input中
可以查看，大多数为配置文件
bin/hdfs dfs -ls /user/hadoop/input/*

#14.运行案例
bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-2.6.0-cdh5.16.2.jar grep input output ‘dfs[a-z.]+’

#15.将hdfs中的output下载到linux中
bin/hdfs dfs -get output output

我们可以看一下
cd output
ll
-rw-r–r– 1 hadoop hadoop 281 Nov 22 18:38 part-r-00000
-rw-r–r– 1 hadoop hadoop 0 Nov 22 18:38 _SUCCESS

其中_SUCCESS是一个标识符
part-r-00000说明我们的结果为一份

cat part-r-00000
6 dfs.audit.logger
4 dfs.class
3 dfs.logger
3 dfs.server.namenode.
2 dfs.audit.log.maxbackupindex
2 dfs.period
2 dfs.audit.log.maxfilesize
1 dfsmetrics.log
1 dfsadmin
1 dfs.servers
1 dfs.replication
1 dfs.namenode.secondary.https
1 dfs.log
1 dfs.file
1 dfs.namenode.secondary.http
看来我们刚才运行的是一个词频统计的程序
