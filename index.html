不积跬步无以至千里不积小流无以成江河 也许很多Linux命令较为抽象不易记忆，但是好记性不如烂笔头，记下来加深记忆，随时补充。 Linux常用命令pwd全称：Print Working Directory查看当前光标所在目录(路径 path) 1$ pwd 结果为绝对路径 ls显示的文件夹和文件的名称 12$ ls -l$ ll 可以看到权限 用户用户组 时间 大小等额外信息 123$ ls -l -a$ ll -a$ ls -la 可以显示隐藏文件夹和文件(隐藏文件夹和文件一般为.开头) 123$ ls -l -h$ ll -h$ ls -lh 仅仅查看文件大小，文件夹不行 123$ ls -l -r -t$ ll -rt$ ls -lrt 按时间升序显示-r 反转排序顺序-t 按时间排序(默认降序) mkdir创建文件夹 1$ mkdir 1 2 3 并联创建文件夹 1$ mkdir -p 1/2/3 并联创建文件夹，此处-p为递归创建目录 cd全称：Change Directory切换目录(路径 path) 1$ cd 绝对路径 或 相对路径 切换目标路径 123$ cd$ cd ...
