HDFS API - 初识

发表于2021-02-27|更新于2021-02-27
|阅读量:

pom.xml

在调用HDFS API的时候，需要很多jar包的支持，所以在pom.xml配置一下

repository

建议在pom.xml中都配置这两个代码
aliyun maven : 这个使很多jar包下载的快一点
cloudera : 如果使用的hadoop是CDH版本的，则必须配置这个，否则找不到对应的jar包

dependency

想要操作HDFS API，hadoop-client这个配置也是必不可少的，我这里使用的是2.6.0-cdh5.16.2版本

另外我们在写代码的时候会进行测试，juint非常好用

FileSystem

使用 @Test 初识 HDFS API

详细的代码解释写在了备注中
这里mkdir()和copyFromLocalFile()的代码重复率很高，配置configuration，设置fileSystem，释放资源等
我们可以使用junit中的 @Before 和 @After 来处理这些重复的内容

@Before 和 @After

使用 @Before 来定义configuration和fileSystem
使用 @After 来释放资源

fileSystem.mkdir() : 在hdfs上创建文件夹
fileSystem.copyFromLocalFile() : 从本地复制文件到hdfs
fileSystem.copyToLocalFile() : 从hdfs复制文件到本地，会生成两个文件，一个是文件本身，一个是.crc数据校验文件
fileSystem.rename() : 重命名hdfs上的文件
fileSystem.delete() : 删除hdfs上的文件/文件夹
fileSystem.listFiles() : 获取指定hdfs文件夹下的文件的一些属性，包括但不限于权限，文件大小，文件个数，地址等

HDFS API - IO操作

还可以使用 org.apache.hadoop.io.IOUtils ，使用IO操作来上传下载HDFS的文件，例如：
copyFromLocalFile的第二种方式 : fileSystem.create
copyToLocalFile的第二种方式 : fileSystem.open
