优势洗牌

难度：中等
网址：https://leetcode-cn.com/problems/advantage-shuffle/
知识点：田忌赛马 Array ArrayBuffer break
感受：写完才发现是田忌赛马，希望在看到这种题的时候不会忘记
Array添加元素可以使用 :+
ArrayBuffer删除元素可以使用remove()
scala中使用break需要导包：import scala.util.control.Breaks._

模糊坐标

难度：中等
网址：https://leetcode-cn.com/problems/ambiguous-coordinates/
知识点：List ListBuffer toInt toDouble asInstanceOf
感受：发现了toInt和asInstanceOf[Int]竟然是有区别的
“00”.toInt 结果是 0
“00”.asInstanceOf[Int] 结果是报错

“0.0001”.toDouble 结果是 1.0E-4 变成科学计数法了 淦
可以用 BigDecimal(“0.0001”) 结果是 0.0001

不过BigDecimal()也有问题
BigDecimal(“00.00”) 结果是 0.00
而 “00.00”.toDouble 结果是 0.0

ListBuffer添加元素可以使用 append() 也可以使用 +=
判断ListBuffer是否有值可以使用 ListBuffer.nonEmpty，当然ListBuffer.size = 0
