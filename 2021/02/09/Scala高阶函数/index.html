Scala高阶函数

发表于2021-02-09|更新于2021-03-06|Scala
|阅读量:

上回书说到
Scala中，函数为头等公民
其中有一些比较重要的头等公民

map

map应该是Scala中最常用的函数了
作用是最每个元素做统一处理
需要注意处理的元素的类型
例如List就是处理List中每个元素，List(0)，List(1)…
String就是处理字符串中每个字符等

foreach

一般配合其他函数使用，常用的就是打印数据
这里注意foreach返回的是Unit类型

filter

意为 过滤
filter() 括号内为boolean类型

flatten

意为 拍平

flatMap

flatMap = flatten + map

reduce

意为 聚合，规约

fold

fold 有两个参数
第一个参数为初始值
第二个参数是运算

zip

zip zipAll zipWithIndex
各有用途
zip 类似于 inner join
zipAll 类似于 full join
zipWithIndex 类似于把行号带出来，不过是从0开始

groupBy

意为 分组

sort

sorted 按数字或者字典表升序
sortBy 自定义排序，可定义多个排序规则
sortWith 两两比较

注意 Map中没有sortBy方法
