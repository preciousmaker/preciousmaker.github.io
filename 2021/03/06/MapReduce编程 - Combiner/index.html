MapReduce编程 - Combiner

发表于2021-03-06|更新于2021-03-06|MapReduce
|阅读量:

#简介
Combiner(预聚合) : 介于Map和Reduce之间的一个Reduce操作，所以其父类是Reducer，但是是运行在MapTask
作用 : 减少数据的网络传输，是一个非常重要的调优点

例如 :
MapTask1 : (a,1),(a,1),(a,1),(b,1)
MapTask2 : (a,1),(a,1),(b,1)
经过Shuffle后有两个reduce，那么
ReduceTask1 : (a,1,1,1,1,1)
ReduceTask2 : (b,1,1)

如果使用了Combiner
MapTask1 : (a,1),(a,1),(a,1),(b,1)
MapTask2 : (a,1),(a,1),(b,1)
经过Combiner
MapTask1 : (a,3),(b,1)
MapTask2 : (a,2),(b,1)
再经过Shuffle
ReduceTask1 : (a,3,2)
ReduceTask2 : (b,1,1)

本来要传输5个1，现在只需要传输3,2，减少数据的网络传输

慎用点 : AVG不适用
